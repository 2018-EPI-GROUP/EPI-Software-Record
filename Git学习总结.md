2.使用

 1.创建空目录，然后使它成为一个本地库
  mkdir <文件名>
  cd  <文件名>
  pwd/目录位置
2.使这个库可管理
  git init
3.添加文件
  git add <文件名>
4.将文件提交到本地库
  git commit -m"对文件的描述"
5.创建并切换分支
  git checkout -b <分支名> 
6.切换到某分支
  git checkout <分支名> 
7.查看当前分支
   git branch 
8.合并分支到主分支
   git merge <分支名> 
9.删除分支
   git branch -d <分支名> 
10.查看库中的信息
   git status
11.查看改动
   git diff 
12.查看提交历史，回到过去的文件
   git log 
   查看命令历史，回到未来的文件
   git reflog 
   转到某个版本
   git reset --hard commit_id 
13.删除该文件
   rm <文件名> 
14.查看该库中的内容
   cd <库名>
   ls  
15.查看文件内容
   cat <文件名> 
16.创建文件
   touch <文件名> 
17.打开某文件并进行编辑，如果该文件不存在，则创建一个
   vim <文件路径或文件名>
